# סוניה ואמיר פלוט — שלוש תיקיות עצמאיות

כל תיקייה היא אתר סטטי עצמאי ל-GitHub Pages וכוללת `index.html`, `gallery.html`, קבצי CSS/JS, תמונות WebP ומסמכי PDF.

## המלצת כיוון
על בסיס ההערות האחרונות, התיקייה `02-dark-warm` מסומנת ככיוון המומלץ: עיצוב כהה שאינו שחור, עם פתיחה קרובה יותר למבנה של אופציית הדארק המקורית ושאר עמוד פשוט ומדויק יותר.

## מבנה התוכן
- רצועת זיכרון
- סימני דרך / אבני דרך — רק באפשרויות 1–2
- דברי פרידה
- רגעים שנשמרו + קישור לגלריה מלאה

הטקסטים נשמרו קצרים ומבוססי מידע שנמסר, כדי להימנע מהרחבות לא מאושרות.
